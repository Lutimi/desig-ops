## Colors


| Role           | Color   | Usage                          |
| -------------- | ------- | ------------------------------ |
| Primary        | #2563EB | Main buttons, links, accents   |
| Secondary      | #7C3AED | Highlights, secondary actions  |
| Accent         | #F59E0B | Call-to-action, alerts         |
| Dark BG        | #0F172A | Dark backgrounds, hero section |
| Mid BG         | #1E293B | Cards, secondary backgrounds   |
| Light BG       | #F8FAFC | Light backgrounds              |
| Text Primary   | #F1F5F9 | Main text on dark backgrounds  |
| Text Secondary | #94A3B8 | Muted text, captions           |

## Logo

- Place your logo file in this folder as `logo.png` or `logo.svg`
- Minimum clear space: 16px around the logo
- On dark backgrounds, use the light version
- On light backgrounds, use the dark version

## Tone & Voice

- Professional but approachable
- Confident, not arrogant
- Clear and concise

## Icons

- Style: Outlined, 2px stroke
- Size: 24x24px base
- Source: Lucide Icons, Heroicons, or Phosphor Icons
